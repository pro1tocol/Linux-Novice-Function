# Environment switch

---------------------------

## Print the current environment
    dpkg --print-architecture
## Switch
    dpkg --add-architecture amd64
    dpkg --add-architecture i386
    dpkg --add-architecture arm64
## Delete environment
    dpkg --remove-architecture i915
`reboot`take effect
