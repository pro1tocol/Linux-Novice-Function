# Chinese command mode
## Select regional language

    local
## List all languages

    local -a
## Download Chinese Language Pack

    local-gen zh_CN.UTF-8
Edit configuration file

    vim /etc/profile
    export LANG='zh_CN.UTF-8'
## Install

    zhcon
## RUN

    zhcon --utf8
