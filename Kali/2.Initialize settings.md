## Program management
`apt`instructions

    apt update              //Update sources
    apt upgrade             //Update system
    apt install             //Construct
    apt remove              //Uninstall
    apt autoremove          //Uninstall automatically
    apt purge               //*Uninstall
    apt list --upgradable   //View to-be-updated list
 `dpkg`instructions
 
    dpkg -i                 //Construct
    dpkg -r                 //Uninstall
`rpm`instructions

    rpm                 //Construct
    rpm -e              //Uninstall
